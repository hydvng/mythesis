## 🔍 发散原因分析

### Stage 3发散 (Episode 340+)

**可能原因:**
1. **学习率过高**: 1e-3的critic学习率对30秒回合可能过高
2. **探索噪声**: 后期探索导致进入状态空间未覆盖区域
3. **数值不稳定**: 长回合累积梯度导致更新不稳定
4. **环境难度**: 30秒回合累积误差更大

**解决方案建议:**
- 降低学习率: critic 1e-3 → 5e-4
- 增加梯度裁剪: 1.0 → 0.5
- 减少探索: 后期减少噪声
- 使用早停: Episode 300后停止
