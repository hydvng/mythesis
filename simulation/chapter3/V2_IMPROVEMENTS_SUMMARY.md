# 第3章强化学习训练改进总结

## 原始问题诊断

在初始训练中发现以下关键问题：

### 1. 动作空间范围不合理
- **问题**: RL输出被限制在±10000N，但环境期望归一化动作[-1, 1]
- **影响**: 训练不稳定，动作尺度不匹配
- **修复**: 设置`action_scale=5000`，动作空间改为[-1, 1]

### 2. 状态归一化溢出
- **问题**: 扰动下速度>0.2，归一化后超出[-1,1]
- **影响**: 状态值超出范围，网络输入不稳定
- **修复**: 调整状态归一化因子

### 3. 摩擦力不连续
- **问题**: 使用`np.sign()`在零速度处跳跃
- **影响**: 训练不稳定，梯度估计困难
- **修复**: 使用`tanh()`光滑近似

### 4. 初始状态设置错误
- **问题**: z初始值0.5-0.56，但约束要求z≥0.808
- **影响**: 所有回合一开始就违反约束
- **修复**: 初始z改为0.9-1.1（约束范围内）

### 5. 期望轨迹设置错误
- **问题**: z_des=0.529（违反约束z_min=0.808）
- **影响**: 控制器被迫违反约束跟踪轨迹
- **修复**: 期望z改为1.0（约束中心）
