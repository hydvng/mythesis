# 扰动引入方法对比分析

## 对比对象

| 项目 | 我的实现 | 3dofPlatformSim项目 |
|------|---------|-------------------|
| **文件** | `common/wave_disturbance.py` | `disturbances/MSS.py` + 说明文档 |
| **谱类型** | JONSWAP谱 | ITTC双参数谱 |
| **物理链路** | 频谱 → 直接合成力 | 波面 → RAO响应 → 力 |

---

## 1. 我的实现方法（当前）

### 代码逻辑
```python
# 1. 生成JONSWAP谱
S = jonswap_spectrum(omega)

# 2. 多频率分量叠加（随机相位）
for each frequency:
    disturbance += amplitude * cos(omega*t + phase)

# 3. 直接缩放为力/力矩
disturbance[:, 0] *= wave_height * 100  # 升沉力
disturbance[:, 1] *= wave_height * 50   # 横摇力矩
disturbance[:, 2] *= wave_height * 50   # 纵摇力矩
```

### 问题
❌ **缺乏物理意义明确的波面→力的映射**
- 直接通过经验系数（100, 50）缩放，没有理论依据
- 没有考虑平台的频响特性
- 不同海况下的扰动特性无法准确区分

❌ **JONSWAP vs ITTC选择**
- JONSWAP更适合深 sea state 描述
- ITTC双参数谱更适合船舶/平台工程应用（有明确的Hs和T1）

---

## 2. 3dofPlatformSim项目方法（更科学）

### 核心思想
**波面过程 η(t) → RAO幅频响应 |H(ω)| → 广义力/力矩 τ(t)**

### 物理链路（清晰完整）

```
海况等级(Sea State) 
    ↓
ITTC双参数谱 S(ω) = 173*Hs²/(T1⁴*ω⁵) * exp(-691/(T1⁴*ω⁴))
    ↓
波面过程 η(t) = Σ A_i * cos(ω_i*t + φ_i)
    ↓
二阶响应函数 |H(ω)| = gain * ωn² / √((ωn²-ω²)² + (2ζωnω)²)
    ↓
广义力/力矩 τ(t) = Σ |H(ω_i)| * A_i * cos(ω_i*t + φ_i)
```

### 关键参数（物理意义明确）

| 参数 | 物理意义 | 单位 | 作用 |
|------|---------|------|------|
| **Hs** | 有义波高 | m | 控制扰动强度 |
| **T1** | 特征周期 | s | 控制扰动频段 |
| **gain** | 增益系数 | N/m 或 N·m/m | 波面→力的转换比例 |
| **ωn** | 固有频率 | rad/s | 响应峰值频率 |
| **ζ** | 阻尼比 | - | 峰值尖锐程度 |

### 优势

✅ **物理意义清晰**
- 先合成波面（m），再通过响应函数转为力（N）
- 符合水动力学基本流程

✅ **可解释性强**
- 每个参数都有明确的物理意义
- 可以通过实验数据拟合gain、ωn、ζ

✅ **可复现性好**
- 使用随机种子保证可复现
- 支持从文件加载预生成的扰动

✅ **与MSS对齐**
- 可以直接使用MSS导出的tau_wave1
- 支持6DOF到3DOF的映射（Z→Fz, K→Mα, M→Mβ）

## 3. 两种方法对比表

| 特性 | 我的方法 | 3dofPlatformSim方法 | 评价 |
|------|---------|-------------------|------|
| **谱类型** | JONSWAP | ITTC | ITTC更适合船舶工程 |
| **海况定义** | Hs, Tp（峰值周期） | Hs, T1（特征周期） | T1更常用 |
| **波面生成** | ❌ 无 | ✅ 有 | 波面是中间物理量 |
| **响应函数** | ❌ 无 | ✅ 二阶系统 | 模拟平台频响 |
| **参数可配** | ❌ 固定缩放 | ✅ gain/ωn/ζ可调 | 更灵活 |
| **MSS兼容** | ❌ 不支持 | ✅ 完全支持 | 便于实验对比 |
| **on_ship模式** | ❌ 不支持 | ✅ 支持 | 考虑船体运动传递 |

## 4. 建议改进方案

### 方案A：直接采用3dofPlatformSim方法（推荐）

**优点**：
- 物理意义明确，符合水动力学
- 已有成熟的代码实现
- 与MSS完全兼容

**实现步骤**：
1. 使用ITTC谱替代JONSWAP谱
2. 添加波面生成步骤
3. 添加二阶响应函数
4. 配置gain、ωn、ζ参数

**参考代码**：
```python
# 1. ITTC谱
S_eta = 173*Hs**2/(T1**4*omega**5) * np.exp(-691/(T1**4*omega**4))

# 2. 合成波面
eta = np.zeros(N)
for w, s, phi in zip(omega, S_eta, phases):
    eta += 2*np.sqrt(s*2*np.pi*freq_res) * np.cos(w*t + phi)

# 3. 响应函数（二阶系统）
|H(w)| = gain * wn**2 / np.sqrt((wn**2-w**2)**2 + (2*zeta*wn*w)**2)

# 4. 生成扰动力
tau = np.zeros(N)
for w, s, phi, h in zip(omega, S_eta, phases, |H(w)|):
    tau += h * 2*np.sqrt(s*2*np.pi*freq_res) * np.cos(w*t + phi)
```

### 方案B：改进当前的JONSWAP方法

**改进点**：
1. 添加波面生成（中间步骤）
2. 添加简化的响应函数
3. 使用更合理的缩放系数

**缺点**：
- JONSWAP谱在船舶领域不如ITTC常用
- 仍缺乏与MSS的兼容性

## 5. 具体实施建议

**推荐做法**：

1. **直接使用3dofPlatformSim的`MSS.py`模块**
   - 该模块已经非常成熟
   - 包含了完整的文档和示例
   - 支持多种使用场景

2. **或者按照以下参数配置**：

```python
# Sea State 4 (中等海况)
sea_state_params = {
    "Hs": 2.0,      # 有义波高 2m
    "T1": 8.0,      # 特征周期 8s
}

# 响应函数参数（大尺度平台：m=347.54kg, Ixx=60.64, Iyy=115.4）
rao_params = {
    "Fz": {"gain": 20000.0, "wn": 1.0, "zeta": 0.15},   # 升沉（gain×10）
    "Mα": {"gain": 320000.0, "wn": 0.9, "zeta": 0.20},  # 横摇（gain×40）
    "Mβ": {"gain": 320000.0, "wn": 0.9, "zeta": 0.20},  # 纵摇（gain×40）
}
```

3. **参数调谐建议**：
   - **扰动太小**：增大gain
   - **某个频段振荡太厉害**：增大zeta或调整wn
   - **低频成分太多**：降低wn
