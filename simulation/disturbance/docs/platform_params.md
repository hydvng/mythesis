## 1. 平台参数说明

### 1.1 关键声明

**本文采用的计算方法为：实际平台在实际船舶上的真实扰动计算**

- **平台质量**: 347.54 kg —— **实际尺寸，非缩尺模型**
- **船舶数据**: MSS Supply船（Lpp=82.8m）—— **实际船舶，非模型**
- **计算方法**: 直接使用MSS实船运动RAO数据，计算实际平台的惯性力响应
- **物理场景**: 一艘82.8m的供应船上装载一个347kg的稳定平台

**重要区分：**
- ❌ 不是"缩尺模型试验"（无需相似变换）
- ❌ 不是"等效模拟"（不是用实船数据代替模型）

## 2. 扰动计算方法说明

### 2.1 方法定义

**本文采用的扰动计算方法：**

```
物理场景：实际船舶 + 实际平台

MSS Supply船（实际船舶）              船载稳定平台（实际平台）
├─ Lpp = 82.8m                         ├─ m = 347.54 kg（实际质量）
├─ 排水量 = 6362吨                      ├─ 尺寸 = 1.16m直径
├─ MSS RAO数据（实船船模试验）          └─ 安装在该船上
└─ 海况：Hs=2.0m, T1=8.0s（实际海况）

计算流程：
  1. ITTC波谱 → 波面 η(t)
  2. MSS运动RAO → 船体运动 X(ω) = RAO(ω) × η(ω)
  3. 运动学 → 加速度 a = ω² × X
  4. 牛顿定律 → 惯性力 F = m_platform × a
```

**关键特征：**
- ✅ 使用MSS实船运动RAO数据（真实船舶响应特性）
- ✅ 平台质量为实际设计值（347.54kg）
- ✅ 计算结果为**实际平台在实际船舶上的真实惯性力响应**
- ❌ **无需相似变换**（非缩尺模型）
- ❌ **非等效模拟**（不是用实船数据代替模型）

### 2.2 与缩比模型的区别

| 对比项 | 缩比模型（1:5） | 本文方法 | 说明 |
|--------|----------------|----------|------|
| **船舶尺寸** | 模型船（16.5m） | 实船（82.8m） | 本文用真实船舶 |
| **平台质量** | 40kg（λ³缩放） | **347.54kg（实际）** | 本文为实际质量 |
| **频率变换** | ω × 2.236 | ω（不变） | 本文直接使用实船频率 |
| **运动幅值** | X × 0.2 | X（不变） | 本文计算真实运动 |
| **物理意义** | 预报实船性能 | **计算实际响应** | 目的不同 |
| **适用场景** | 水池模型试验 | **实船装载设计** | 本文适用 |

**本文方法的优势：**
1. **直接性**：无需相似变换，计算简洁明了
2. **真实性**：基于实船RAO数据，工程可信度高
3. **实用性**：直接指导实际平台设计（347kg级）
4. **明确性**：物理图像清晰，无尺度歧义

### 2.3 扰动参数影响

#### 物理关系

惯性力公式：$F = m \cdot a$

- 质量增加10倍 → 惯性力增加10倍
- 扰动绝对值：4.6 N → **45.9 N**
- 扰动占比：**保持1.35%不变**（重力也增加10倍）

## 3. 配置验证

### 3.1 作动力余量分析

```
平台重力: 3409 N
扰动力: 46 N (海况4)
作动力上限: 30000 N

余量 = 30000 - 3409 - 46 ≈ 26545 N
余量倍数 = 26545 / 3409 ≈ 7.8 倍
```

✅ **非常充足！** 可轻松补偿重力、扰动和动态加速度。

### 3.2 速度限制合理性

- 速度限制: **0.2 m/s** (12 m/min)
- 典型船载平台运动速度: 0.05-0.3 m/s
- 该限制确保运动平稳，减少冲击
